# RoomSync - Finding Room Easily
Finding a rental room is often stressful and time-consuming. Students, working professionals, and families usually have to search through multiple websites or social media groups, contact several owners, and visit different locations before finding the right place. At the same time, room owners struggle to reach genuine tenants and repeatedly answer the same questions about their properties. This inspired me to build StayBridge, a platform that makes the room rental process simple, transparent, and convenient for both owners and renters.

StayBridge is a full-stack room rental platform that connects room owners with people looking for rental accommodations. Room owners can create listings by adding details such as rent, room type, amenities, location, availability, photos, and videos, while also managing booking requests and updating property status. Renters can search for rooms using filters like city, price, room type, Wi-Fi, AC, parking, furnished rooms, and more. They can view complete property details, ratings, reviews from previous tenants, and book rooms online by paying an advance amount through a secure payment system. The platform also includes features like authentication, personalized dashboards, booking history, wishlists, notifications, and a responsive user interface.

This project is built using modern web technologies and helped me gain hands-on experience in frontend development, backend integration, authentication, database management, payment integration, media handling, responsive UI design, and building a scalable full-stack web application that solves a real-world problem.
